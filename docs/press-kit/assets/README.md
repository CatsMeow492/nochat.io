# Press Kit Brand Assets

Place the following brand assets here for press use:

## Required Assets

### Logo
- `logo.svg` - Vector logo (primary)
- `logo-dark.png` - Logo for dark backgrounds (1024x1024)
- `logo-light.png` - Logo for light backgrounds (1024x1024)
- `logo-horizontal.svg` - Horizontal lockup with wordmark

### App Icon
- `icon.svg` - Vector app icon
- `icon-512.png` - App icon 512x512
- `icon-1024.png` - App icon 1024x1024 (App Store)

### Social
- `og-image.png` - Open Graph image (1200x630)
- `twitter-card.png` - Twitter card image (1200x628)

### Banners
- `banner-wide.png` - Wide banner (1920x600)
- `banner-square.png` - Square banner (1080x1080)

## Brand Colors

| Name | Value | Usage |
|------|-------|-------|
| Primary | `oklch(0.75 0.18 156.17)` | Buttons, accents |
| Background | `oklch(0.145 0 0)` | Dark theme base |
| Surface | `oklch(0.205 0 0)` | Cards, inputs |
| Text | `oklch(0.985 0 0)` | Primary text |
| Muted | `oklch(0.708 0 0)` | Secondary text |

## Typography

- **Primary Font**: System font stack
- **Headings**: Bold weight
- **Body**: Regular weight

## Usage Guidelines

- Maintain aspect ratio of logos
- Don't modify colors or proportions
- Use appropriate contrast backgrounds
- Credit as "NoChat" (not "nochat" or "No Chat")

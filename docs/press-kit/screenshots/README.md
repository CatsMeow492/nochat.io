# Press Kit Screenshots

Place the following screenshots here for press use:

## Required Screenshots

1. **landing-page.png** (1920x1080)
   - Hero section of nochat.io
   - Desktop browser view

2. **chat-mobile.png** (390x844 or similar)
   - Mobile chat interface
   - Show encryption lock icon
   - Active conversation with messages

3. **video-call.png** (1920x1080 or mobile)
   - Two users in encrypted video call
   - Show call controls

4. **anonymous-mode.png** (mobile)
   - "Start Meeting" or anonymous access flow
   - No signup required

5. **security-page.png** (1920x1080)
   - /security page showing crypto details

6. **cross-platform.png** (composite)
   - Side-by-side: Web, iOS, Android
   - Same conversation across devices

## Screenshot Guidelines

- Use real (or realistic demo) conversations
- Don't include real user data or PII
- Ensure encryption indicators are visible
- Dark mode preferred (matches brand)
- High resolution (2x for retina)

## Naming Convention

`{description}-{platform}-{size}.png`

Examples:
- `chat-mobile-390x844.png`
- `landing-desktop-1920x1080.png`
- `video-call-composite-1920x1080.png`
